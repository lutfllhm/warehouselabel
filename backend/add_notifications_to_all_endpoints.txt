# Instruksi untuk menambahkan logging notifikasi ke semua endpoint

Tambahkan logging notifikasi ke endpoint berikut dengan pattern yang sama:

## Label Stocks
- POST /api/label-stocks → "menambahkan stock label"
- PUT /api/label-stocks/:id → "mengupdate stock label"
- DELETE /api/label-stocks/:id → "menghapus stock label"

## Categories
- POST /api/categories → "menambahkan kategori"
- PUT /api/categories/:id → "mengupdate kategori"
- DELETE /api/categories/:id → "menghapus kategori"

## Transactions In
- POST /api/transactions/in → "menambahkan transaksi label masuk"
- PUT /api/transactions/in/:id → "mengupdate transaksi label masuk"
- DELETE /api/transactions/in/:id → "menghapus transaksi label masuk"

## Transactions Out
- POST /api/transactions/out → "menambahkan transaksi label keluar"
- PUT /api/transactions/out/:id → "mengupdate transaksi label keluar"
- DELETE /api/transactions/out/:id → "menghapus transaksi label keluar"

## Documents
- POST /api/documents/lps → "menambahkan dokumen LPS"
- POST /api/documents/sj → "menambahkan dokumen SJ"

## Users
- POST /api/users → "menambahkan user baru"
- PUT /api/users/:id → "mengupdate data user"
- DELETE /api/users/:id → "menghapus user"

Pattern:
```javascript
const username = req.user?.username || "Unknown";
await createNotification(
  req.user?.id || 0,
  username,
  "create|update|delete",
  "Entity Type",
  entityName,
  `${username} [action] [entity]`
);
```
